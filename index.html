Admin nastavenie
Zadaj GitHub token a PIN pre admin prístup. Toto nastavenie sa uloží len v tvojom prehliadači — nováčik nič vypĺňať nemusí.
1
Choď na github.com/settings/tokens
2
Klikni Generate new token (classic)
3
Zaškrtni scope repo a vygeneruj token
4
Skopíruj token a vlož ho nižšie
Token sa uloží len v tvojom prehliadači. Nováčik ho nikdy nevidí ani nepotrebuje.
Zadaj admin PIN
Pre prístup k úpravám zadaj PIN.
